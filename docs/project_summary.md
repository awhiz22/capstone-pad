
# Project Summary

## Problem Statement
What factors drive home sale prices in Ames Iowa and 
how can buyers and sellers use this information to 
make better decisions?

## Dataset
Ames Housing Dataset — 2930 rows and 75 columns

## Key Insights
- Above ground living area has the strongest correlation 
  with sale price at 0.71
- Homes with central air sell for almost double 
  compared to homes without it
- Northridge, Stone_Brook and Northridge_Heights are 
  the most expensive neighborhoods

## Recommendations
- Sellers should consider adding central air before listing
- Buyers looking for value should target larger homes 
  in lower priced neighborhoods

## Limitations
- Correlation does not prove causation
- Dataset covers only Ames Iowa
- Simple imputation used for missing values
